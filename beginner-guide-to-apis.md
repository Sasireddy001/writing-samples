## What Is an API?

API stands for **Application Programming Interface**.

An API allows one software system to communicate with another.

Think of it like a waiter in a restaurant:
- You (client) place an order
- The waiter (API) takes it to the kitchen
- The kitchen (server) sends the result back

---

## Why Are APIs Important?

APIs are used for:
- Mobile apps
- Web apps
- Payment systems
- Social media integrations
- Data pipelines

Without APIs, systems would not be able to share data easily.
